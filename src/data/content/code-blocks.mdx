## Code Blocks

### Basic Code Blocks

Use triple backticks for multi-line code:

````markdown
```
function hello() {
  console.log("Hello, World!");
}
```
````

**Result:**

```
function hello() {
  console.log("Hello, World!");
}
```

### Indented Code Blocks

Alternative syntax using 4 spaces:

```markdown
    function hello() {
      console.log("Hello, World!");
    }
```

**Result:**

    function hello() {
      console.log("Hello, World!");
    }

## Advanced Code Block Features

### Line Numbers

Some processors support line numbers:

````markdown
```javascript {.line-numbers}
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}
```
````

### Line Highlighting

Highlight specific lines:

````markdown
```javascript {hl_lines=[2,4]}
function calculateTax(amount, rate) {
  const tax = amount * rate; // highlighted
  const total = amount + tax;
  return total; // highlighted
}
```
````

## Best Practices

### 1. Choose Appropriate Languages

Use the correct language identifier:

✅ **Correct:**

````markdown
```javascript
const message = "Hello, World!";
```
````

❌ **Generic:**

````markdown
```
const message = "Hello, World!";
```
````

### 2. Keep Code Readable

Format code properly:

✅ **Well formatted:**

````markdown
```python
def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n - 1) + calculate_fibonacci(n - 2)
```
````

❌ **Poor formatting:**

````markdown
```python
def calculate_fibonacci(n):
if n <= 1:
return n
return calculate_fibonacci(n - 1) + calculate_fibonacci(n - 2)
```
````

### 3. Add Context

Provide context for code examples:

✅ **With context:**

````markdown

app.get("/", (req, res) => {
````

## Common Mistakes

### 1. Wrong Backtick Count

❌ **Wrong:**

```markdown
`code here`
```

✅ **Correct:**

````markdown
```
code here
```
````

### 2. Missing Language

❌ **No syntax highlighting:**

````markdown
```
function hello() {
  console.log("Hello");
}
```
````

✅ **With syntax highlighting:**

````markdown
```javascript
function hello() {
  console.log("Hello");
}
```
````

### 3. Inconsistent Indentation

❌ **Mixed indentation:**

````markdown
```python
def function():
  if True:
      print("inconsistent")
  else:
    print("indentation")
```
````

✅ **Consistent indentation:**

````markdown
```python
def function():
    if True:
        print("consistent")
    else:
        print("indentation")
```
````
