## Installation

First, check to see if Java is already installed on your computer.  

- Open a Command Window on Windows or a terminal in Linux.
- At the prompt run the command `java -version `
  - If this command fails, you will need to install Java.
  - If this command succeeds, skip the Java Installation steps.
- Java Installation steps
  - In your browser go to page `https://www.oracle.com/java/technologies/downloads/`
  - Click on the latest JDK link in the ***Java ##***
